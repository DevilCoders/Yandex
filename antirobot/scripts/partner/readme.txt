Данный скрипт реализует логику обработки капчи XML-партнерами Яндекса и предназначен для прояснения некоторых тонких моментов. 

1. Перед началом использования скрипта пропишите ваши идентификационные данные в http_serv.py:  
   
   YANDEX_QUERY="/xmlsearch?showmecaptcha=yes&user=partner&key=xml-key"              

2. На машине, IP-адрес которой зарегистрирован в Яндекс.XML , запустите сервер с помощью команды: 

   ./http_serv.py <порт>

3. На машине, с которой вы будете делать запросы, в файл hosts добавьте строку: 

   <IP сервера>             blah-blah.ru

(В Windows это файл %WINDIR%\System32\drivers\etc\hosts, в Linux - /etc/hosts)
   
Вы можете прописать имя другого домена, например,  вашего. В таком случае вам не придется править hosts, однако потребуется указать этот домен в файле http_serv.py  в строке MY_DOMAIN = ".blah-blah.ru". 

4. Теперь вы можете открыть в браузере http://blah-blah.ru:<порт> и делать  запросы.

Сервер будет выставлять куку spravka на указанном домене, поэтому заходить на него по IP-адресу нельзя,  браузер не примет куку.
