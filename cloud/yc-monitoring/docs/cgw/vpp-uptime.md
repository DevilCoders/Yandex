# vpp-uptime

Проверка показывает сколько времени vpp находится в активном состоянии. При этом, если uptime меньше 5 минут, то проверка жёлтая. Также, если инстанс помечен тегом `scheduled-restartable`, то в случае, если vpp больше суток не рестартился, загорается красным, на что возбуждает процесс авторекавери при попадании во временной таймслот.
