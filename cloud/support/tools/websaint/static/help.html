Как пользоваться этим интерфейсом?

По умолчанию отображение запросов на повышение квот осуществляется по Cloud ID. Таким образом мы можем обработать все запросы без хождения по списку тикетов.
Для одобрения квоты тыкаем «Approve», выставляем одобренный лимит.
Для отклонения квоты, т.е. в принципе в этом запросе повышать не планируем, нажимаем «Reject».
Для пропуска квоты в текущей итерации, например для того, чтобы остваить квоту на согласование, тыкаем «Pass».
Проходим по списку квот в запросе. В итоге тыкаем «Send».
В итоге получаем ответ ...-Done.
