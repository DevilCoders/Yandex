v0.2
Добавил:
- Парсинг вывода списка метрик из quotas-linux из формате prettytable в list. для удобства работы с данными.
- Метод human_to_bytes который конвертирует значения квот из формата human_readable типа '8 Tb, 664 Gb' в байты.
- Убраны ANSII коды цветов при чтении из таблиц quotas-linux.  Коды цветов мешали обработке данных и фэйлили скрипт.
- Добавлена проверка корректности формата значения квоты перед конвертацией в байты и обработано при помощи exception. Эта проверка облегчит поиск проблемы в будущем, если изменится формат вывода в quotas-linux. Вместо непонятного trace  будет выводится человеческая ошибка.
- Актуализирован список названий квот compute, требующих конвертации.
- Сделан MVP. Пока работает стабильно только с compute
