#### IAM All Users Table:

Последний известный снимок таблицы пользователей в IAM

##### Схема

[PROD](https://yt.yandex-team.ru/hahn/navigation?path=//home/cloud-dwh/data/prod/ods/iam/users)
/ [PREPROD](https://yt.yandex-team.ru/hahn/navigation?path=//home/cloud-dwh/data/preprod/ods/iam/users)

- `iam_uid` - идентификатор пользователя в IAM
- `subject_type` - тип пользователя
- `subject_cloud_id` - облако в котором создан пользователь (актуально для SA)
- `created_at` - дата создания
- `modified_at` - дата модификации
