  • Раздача заданий

    Основные принципы:
    1. Задания раздаются комплектами
    2. Асессор может взять комплект либо проигнорировать его
    3. Оплачивается только полностью выполненный комплект заданий

    Как взять задания в оценку

    1. Открываем пункт меню "Оценка" → "Задания"

    1. Открываем вкладку "Доступные". На странице выводятся доступные комплекты заданий с количественной статистикой
    2. Кликаем на кнопку "Взять комплект"

    1. Переходим на вкладку "Текущие"
    2. Кликаем на иконку "+" для разворачивания списка заданий
    3. Выбираем задание из списка, кликаем на него

  • Процесс оценки

    В процессе оценки все критерии теперь оцениваются на отдельных страницах/шагах:

    Шаг 1. Читаемость

    1. При оценке читаемости информация о запросе, регионе, url не выводится
    2. Также не выводится информация о прокраске

    Шаг 2. Содержательность

    1. При оценке содержательности по-прежнему не выводится информация о запросе, регионе, прокраске заголовка и тела сниппета, но показывается url документа

    Шаг 3. Информативность


    При оценке информативности выводится:
    1. Информация о запросе и регионе
    2. Прокраска слов запроса в теле и заголовке сниппета, а также url документа

    • Task distribution

      Basic principles:
      1. Tasks are distributed in sets named as "Task packs"
      2. An assessor can take a "Task pack" or ignore it
      3. An assessor gets paid only for fully estimated set of tasks

      How to get tasks

      1. Open the "Estimations → Tasks" menu item

      1. Go to the "Available" tab. Available task packs with task quantity are displayed on this page
      2. Click the "Take pack" button to the right of the selected pack.

      1. Go to the "Current" tab
      2. Click the "+" icon to expand the list of tasks
      3. Click on any task from this list

    • Assessment process

      During the assessment process each criterion is estimated on its own page:

      Step 1. Readability

      1. When the snippets' readability is being estimated, the query, the region and the document url are not displayed
      2. The query words highlighting is also not displayed

      Step 2. Content-richness

      1. When the snippets' content-richness is being estimated, the query, the region and the query words highlighting are not displayed. But the document url is displayed at this step

      Step 3. Informativity


      When the snippets' informativity is being estimated, the following information is displayed:
      1. The query, the region, the document url
      2. The query words highlighting
