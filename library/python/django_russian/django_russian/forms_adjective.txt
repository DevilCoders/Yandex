# словарик для склонения прилагательных относительно числительных
# Формат: 
#
# - строки с окончаниями, разделенными TAB'ами
# - первое окончание -- именительный падеж, единственное число ("забаненный")
# - остальные два -- снова единственное и множественное число ("забавненных")
#
# Более специфичные варианты должны идти раньше общих
#
# Комментарии начинаются с "#", пустые строки допустимы

#	ед	ед	мн
ый	ый	ых
ая	ая	ых
ее	ее	их
ий	ий	их
