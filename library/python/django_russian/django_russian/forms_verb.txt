# словарик для спряжения глаголов относительно числительных
# Формат: 
#
# - строки с окончаниями, разделенными TAB'ами
# - первое окончание -- единственное число ("режет")
# - остальные два -- снова единственное и множественное число ("режут")
#
# Более специфичные варианты должны идти раньше общих ("держать" раньше "ать")
#
# Комментарии начинаются с "#", пустые строки допустимы

#	ед	ед	мн
