#
# - строки с окончаниями, разделенными TAB'ами
# - первое окончание -- именительный падеж, единственное число ("штука")
# - остальные три -- окончания для количеств 25 ("штук"), 21 ("штука") и 4 ("штуки")
#
# Более специфичные варианты должны идти раньше общих ("поток" раньше "ок")
#
# Комментарии начинаются с "#", пустые строки допустимы
